# 发行版模板

发布 GitHub Release 时使用，复制下方内容填入发行版标题和描述。

---

## 标题

```
栾媛小工具 v1.0.0
```

---

## 内容

```markdown
## 栾媛小工具 v1.0 - 首版发布

基于 Electron + Vue 3 + Element Plus 的桌面小工具集，采用插件化架构。

### 内置插件
- **端口查杀**：根据端口查询并终止占用进程
- **图标生成**：文字生成 ICO 图标，自定义样式与多尺寸
- **秀恩爱**：纪念日记录与展示
- **养龙虾**：趣味桌面龙虾小游戏
- **加减混合**：口算题生成器，可导出 Excel 与打印
- **锁屏（关灯）**：全屏黑屏 + 大眼睛跟随鼠标
- **锁屏（win更新）**：Windows 更新风格假锁屏
- **屏保（碎屏）**：碎屏效果桌面屏保

### 说明
- 主程序构建不包含插件，插件由用户通过插件市场安装
- 插件目录：`~/.ly/tools/plugins`
- 完整更新日志见 [CHANGELOG.md](CHANGELOG.md)
```
