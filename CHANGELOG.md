# 更新日志

本文档记录栾媛小工具的版本变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

---

## [1.2.0] - 2026-03-31

### 主程序

- **布局**：主内容区（`app-main`）不再整体滚动，滚动发生在各页面/插件根节点内，避免与插件内部出现双滚动条。
- **插件主进程脚本**：`invoke-plugin-main` 在每次调用前清除目标脚本的 `require` 缓存，更新插件目录内的 `db-sync.js` 等方法后无需重启主程序即可加载新导出（如 `inspectConnection`）。

### 数据库同步插件（v1.1.0）

- **最低主程序版本**：`minLyToolsVersion` 提升至 **1.2.0**（需配合本版主程序能力）。
- **连接详情**：支持在已填写连接时单独探测库信息（`inspectConnection`），不触发同步。
- **界面**：连接配置可折叠；同步按钮与标题同排右对齐；插件内滚动条改为细样式。
- **同步**：延续对 PostgreSQL `jsonb[]` 等数组类型的插入规范化，减少 `malformed array literal` 等问题。

---

## [1.0.0] - 2025-03-22

### 新增

- **主程序**
  - 基于 Electron + Vue 3 + Element Plus 的桌面应用
  - 插件化架构，支持运行时动态加载插件
  - 插件市场入口（可选安装）
  - 托盘最小化、退出

- **内置插件**
  - **端口查杀**：根据端口查询并终止占用进程（Windows/Linux）
  - **图标生成**：文字生成 ICO 图标，自定义样式与多尺寸
  - **秀恩爱**：纪念日记录与展示
  - **养龙虾**：趣味桌面龙虾小游戏
  - **加减混合**：口算题生成器，可导出 Excel 与打印
  - **锁屏（关灯）**：全屏黑屏 + 大眼睛跟随鼠标
  - **锁屏（win更新）**：Windows 更新风格假锁屏
  - **屏保（碎屏）**：碎屏效果桌面屏保

### 已知限制

- 锁屏、屏保、养龙虾等全屏类插件目前主要针对 Windows 64 位优化
- 插件需预构建为 UMD 格式，不支持热更新
- 图标生成使用系统字体，跨平台效果可能略有差异
- 端口查杀在 Linux 下需具备相应权限才能终止进程
